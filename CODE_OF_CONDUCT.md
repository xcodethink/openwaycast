# Contributor Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## In short

We pledge to make participation a harassment-free experience for everyone, regardless of background or identity. Be respectful, assume good faith, accept constructive feedback gracefully, and focus on what's best for the community.

Unacceptable behavior includes harassment, insults, trolling, publishing others' private information, and other conduct that would reasonably be considered inappropriate.

## Enforcement

Report unacceptable behavior privately to the maintainers via the repository's **Security → Report a vulnerability** (for sensitive reports) or by opening an issue for non-sensitive matters. Maintainers will review and respond, and may remove, edit, or reject contributions, or ban contributors, that violate this Code of Conduct.

Full text and enforcement guidelines: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
